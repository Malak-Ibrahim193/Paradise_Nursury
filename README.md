# Paradise Nursery

Paradise Nursery is an online plant shop where users can browse, view details, and add houseplants to their shopping cart.

## Features
- Browse plants by category
- View details including image, name, price
- Add plants to cart with dynamic updates
- Interactive cart management

## Tech Stack
- React.js
- Redux
- CSS

## How to Run
1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `npm start` to start the development server
